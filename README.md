# Number Plate Detection using Python

This project detects vehicle number plates using OpenCV and Tesseract OCR.

## Features

* Detect license plates from images
* Extract text using OCR
* Draw bounding boxes around detected plates

## Technologies Used

* Python
* OpenCV
* pytesseract

## Installation

```bash
pip install opencv-python pytesseract
```

## Run the Project

```bash
python main.py
```
